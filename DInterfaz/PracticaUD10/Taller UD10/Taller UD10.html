WORKSHOP 10 - FLOATING AND POSITIONING -

Exercise 10-1: Floating images

  1. Open the CSS file bakery-styles.css in a text or code editor, and open bakery.html in a browser. We'll start by removing wasted vertical space next to the baked good images by floating those images to the left. Create a new style rule with a contextual selector to target only the images in the main section: 
    main img {
    float: left;
}
  2. We want the "Learn more" links to always appear below the images so they are clearly visible and consistently on the left side of the page. Fortunately, the paragraphs with those links are marked up with the class "more", and there is already a style rule for them using a class selector. Make those paragraphs clear any floats on the left edge: 
    p.more {
    ...
    clear: left;
}
  3. Lastly, we'll adjust the spacing around the floated images. Give both images on the right and bottom sides by using the shorthand margin property: 
    main img {
    float: left;
    margin: 0 16px 16px 0;
}

    The muffin image could use extra space on the left side so it lines up better with the bread. Use this nifty attribute selector to grab any image whose src attribute contains the word muffin (there's only one):

    img[src*="muffin"] {
    margin-left: 50px;
}

Exercise10-2: Adding shapes around floats

  1. Target each image individually using an attribute selector (there is one set up for "muffin" already). Start out simply and make the text wrap around a circle. Set the radius of the circle to 125px for the bread image and 110px for the muffin: 
    img[src*="bread"] {
    shape-outside: circle(125px);
}
img[src*="muffin"] {
    margin-left: 50px;
    shape-outside: circle(110px);
}
  2. Save the styles and take a look at the bakery.html page in a supporting browser. The circles look pretty good, but we could improve the wrap around the bread image by making it an ellipse. Add the new ellipse declaration after the circle declaration, so it overrides the previous style (or delete and replace the circle declaration): 
    img[src*="bread"] {
    shape-outside: circle(125px);
    shape-outside: ellipse(130px 105px at 50% 50%);
}
  3. If you're feeling ambitious, try adding a polygon wrap shape around the muffin image instead of the circle (if not, don't worry about it, nobody will judge you!). You'll need to copy these coordinates manually, or copy and paste them from the materials for this chapter: 
    img[src*="muffin"] {
    ...
    shape-outside: polygon(0px 0px, 197px 0px, 241px 31px, 241px 68px, 226px 82px, 219px 131px, 250px 142px, 250px 158px, 0px 158px);
}

Exercise10-3: Absolute positioning

  1. Good news! Black Goose Bakery won the Farmers' Market Award, and we have the privilege of displaying an award medal on the home page. Because this is new content, we'll need to add it to the markup in bakery.html. Because it is nonessential information, we'll add the image in a new div in the footer of the document: 
    <footer>
    <p>All content &copy; 2024, Black Goose Bistro.</p>
    <div id="award"><img src="images/award.png" alt="Farmers' Market Award"></div>
</footer>
  2. Just because the award image is at the end of the source document doesn't mean it needs to display there. We can use absolute positioning to place the award in the top-left corner of the viewport by adding a new rule to the bakery-styles.css style sheet that positions the div like so (I put mine in the /* misc styles */ section): 
    #award {
    position: absolute;
    top: 30px;
    left: 50px;
}

    Save both files and load the HTML document in your browser. Resize the browser window very narrow, and you will see that the positioned award image overlaps the header content. Notice also that when you scroll the document, the image scrolls with the rest of the page. Try playing around with other offset properties to get a feel for positioning in the viewport (or the initial containing block, to be precise).

    PS: I know that the navigation list still looks bad, but we'll fix it up later.

Exercise10-4: Fixed positioning

This should be simple. Open the style sheet as you left it in EXERCISE10-3 and edit the style rule for the #award div to make its position fixed rather than absolute:

#award {
    position: fixed;
    top: 30px;
    left: 50px;
}

Save the styles and open the page in a browser. When you scroll the page, you will see that the award graphic now stays put where you positioned it in the browser window. You can see that fixed positioned elements have the potential to hide content as the page scrolls. Test well to see the potential pitfalls and weigh them against the benefits.
