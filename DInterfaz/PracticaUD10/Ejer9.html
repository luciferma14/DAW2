Ejercicio 9: Tooltip Inteligente

Objetivo: Crea un tooltip que aparezca al hacer hover, centrado respecto al elemento trigger.

Pasa el ratón sobre esta palabra para ver el tooltip.

También puedes probar con otra palabra en diferentes posiciones del texto.

El tooltip debería aparecer correctamente posicionado sin importar dónde esté la palabra en la página.

Reto:

Hazlo solo con CSS (sin JavaScript).

Requisitos:

  • Debe aparecer centrado respecto al elemento trigger
  • Debe tener una flecha apuntando al elemento
  • Debe aparecer/desaparecer suavemente
  • Solo debe usar CSS (::before y ::after)
